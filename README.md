## Comando para execução:

Obs: deve-se ter o make instalado.

### Windows
```
make; .\LEITOR.exe ./java/<NomeDoArquivo>.class
```

### Linux

```
make; .\LEITOR.exe ./java/<NomeDoArquivo>.class
```

### Usando g++:
```
g++ <nome_arquivo1.cpp> <nome_arquivo2.cpp> ... <nome_arquiboN.coo> -o leitorexibidor

./leitorexibidor
```
